TGP Texture Map Generator

Outil de génération de maps PBR (Normal, Specular, Emissive, Height) à partir d'une texture source, 100% dans le navigateur.

Conçu pour le workflow GTA V / FiveM / OpenIV dans le cadre du projet Lonehaven de TyNox Games Production. Aucune donnée n'est envoyée sur un serveur — tout le traitement se fait localement via l'API Canvas HTML5.

ℹ Information
Compatible avec les textures au format PNG, JPG et WebP. Pour de meilleurs résultats, utilisez des textures en résolution puissance de 2 (512, 1024, 2048…).

Charger une texture

  1. Cliquez sur le bouton + dans la bande de textures, ou glissez-déposez une image directement dessus.
  2. Plusieurs textures peuvent être chargées simultanément — cliquez sur une miniature pour la sélectionner.
  3. La texture active s'affiche dans l'aperçu gauche (SOURCE).
  4. Sélectionnez un type de map dans les onglets, ajustez les paramètres, la map se génère automatiquement.
  5. Téléchargez via ↓ Télécharger map ou via le panneau d'export multiple.

Pour supprimer une texture, survolez sa miniature et cliquez sur le × qui apparaît.

Normal Map _n

La normal map encode la direction de la normale de surface dans les canaux RGB. Elle simule un relief détaillé sur un mesh plat, sans polygones supplémentaires. GTA V utilise le suffixe _n.

L'algorithme applique un filtre de Sobel sur la luminance de la texture source pour détecter les gradients horizontaux (X) et verticaux (Y), puis encode le vecteur normal normalisé :

R = (Nx + 1) / 2    → direction horizontale
G = (Ny + 1) / 2    → direction verticale (↕ selon convention)
B = Nz              → composante profondeur (≈ 0.5–1.0)
Fond neutre = RGB(128, 128, 255)

Paramètres

ParamètrePlageDescription
Strength0.1 – 10Intensité du relief perçu. Plus élevé = angles plus marqués. Valeur recommandée : 2–4 pour de la pierre/béton, 1–2 pour du bois/métal.
Blur (pré-filtre)0 – 5Applique un flou sur la source avant le calcul des gradients. Réduit le bruit mais adoucit les détails fins.
Inverser Y (DirectX)✓ / ✗Inverse le canal vert (G). Cochez pour les moteurs DirectX (OpenIV, FiveM). Décochez pour OpenGL.

Conseils GTA V

✓ Conseil
Pour GTA V / OpenIV, activez Inverser Y. Le moteur RAGE utilise la convention DirectX (canal vert inversé). La couleur neutre doit être RGB(128, 128, 255) — bleu pur = surface plate.
⚠ Attention
Le filtre Sobel fonctionne mieux sur des textures à fort contraste de luminance (bords, reliefs visibles). Sur une texture très plate ou monochrome, la normal map sera quasi-uniforme.

Specular Map _s

La specular map contrôle la réflectivité spéculaire de chaque pixel : blanc = très brillant, noir = mat. GTA V utilise le suffixe _s. Le format est un canal de luminance (grayscale).

Modes

Standard (gradient)

Convertit la luminance de chaque pixel en valeur de spécularité via une courbe gamma.

ParamètreDescription
IntensitéMultiplicateur global de la spécularité.
Gamma (brillance)Courbe de puissance. >1 écrase les tons sombres, <1 ouvre les hautes lumières.
ContrasteÉtire la plage de valeurs autour de 128.
InverserInverse la map (zones sombres = brillantes). Utile pour les matériaux où les zones foncées sont métalliques.

Binaire (GTA V)

Produit une map noir pur / blanc pur à partir d'un seuil de luminance. C'est le format typiquement peint à la main dans les assets GTA V — voir l'exemple 132courtdoor1_s.png.

ParamètreDescription
SeuilLuminance à partir de laquelle un pixel devient blanc (brillant).
Transition douceZone de fondu autour du seuil. À 0 = binaire dur.
InverserLes pixels sombres deviennent brillants. Utilisé quand les zones foncées (ex: verre) doivent être réfléchissantes.
✓ Exemple — porte de tribunal
Pour reproduire 132courtdoor1_s.png : mode Binaire, seuil ~100, Inverser coché (les vitrages sombres → blanc = brillant), transition ~5.

Emissive Map _e

La emissive map définit les zones qui émettent de la lumière indépendamment de l'éclairage de scène. Dans GTA V, elle est utilisée pour les néons, fenêtres allumées, écrans, yeux de personnages.

ParamètreDescription
Seuil de luminositéLes pixels au-dessus de ce seuil de luminance sont considérés émissifs.
Douceur (falloff)Zone de transition progressive autour du seuil. Évite les bords durs.
Couleur de tintTeinte appliquée aux zones émissives (ex: orange pour des fenêtres, bleu pour des néons).
Mix tintForce du mélange entre la couleur source et la couleur de tint. 0 = source pure, 1 = tint pur.
ℹ Usage FiveM
Dans FiveM, l'emissive map fonctionne en combinaison avec le shader emissive sur le drawable. Elle ne remplace pas l'éclairage de scène mais s'y ajoute.

Height / Parallax Map _h

La height map (ou parallax map) encode l'élévation de chaque point de surface en niveaux de gris : blanc = en relief, noir = en creux (convention standard). Elle est utilisée pour le parallax occlusion mapping.

ParamètreDescription
ContrasteÉtire les valeurs de gris pour accentuer les différences de hauteur.
LuminositéDécale globalement la hauteur vers le haut ou le bas.
Lissage (blur)Adoucit la height map, ce qui produit des transitions de hauteur plus douces.
InverserInverse creux et relief. Utile si votre shader attend la convention inverse.

Couleur _color

Permet d'ajuster et de transformer les couleurs de la texture source sans générer une map secondaire. Utile pour créer des variantes de couleur d'un même asset (ex: déclinaisons de couleurs de portes, murs...).

ParamètreDescription
Teinte (hue)Rotation de la teinte dans l'espace HSL, de -180° à +180°. Décale toutes les couleurs.
SaturationIntensité des couleurs. Valeur négative → désaturé (proche du gris).
LuminositéÉclaircit ou assombrit la texture.
ContrasteÉtire la plage tonale autour du gris moyen.
Couleur de tint + OpacitéSuperpose une teinte colorée avec opacité réglable (blend mode: normal).
InverserInverse toutes les couleurs (négatif photographique), appliqué avant les ajustements HSL.

Résolution ⇔

Redimensionne la texture vers une résolution standard ou personnalisée. GTA V requiert des textures en puissance de 2 dans les deux dimensions.

Résolutions courantes GTA V : 64 128 256 512 1024 2048 4096

Algorithmes de qualité

ModeUsage recommandé
PixelatedNearest-neighbor. Aucun flou. Idéal pour les textures pixel-art ou les upscales intentionnellement pixelisés.
LisséBilinéaire haute qualité. Bon pour les upscales modérés (+1 niveau).
Stepped ↓Halvings progressifs. Recommandé pour les downscales importants (ex: 2048→256). Évite les artefacts de sous-échantillonnage.
⚠ Attention
L'upscaling (augmenter la résolution) n'ajoute pas de détails. Il agrandit simplement les pixels existants. Pour améliorer réellement la qualité, utilisez un outil d'upscaling IA dédié.

Export multiple

Le panneau d'export en bas de l'interface permet de sélectionner exactement quelles maps télécharger en une seule opération.

  1. Cochez les maps souhaitées dans le panneau Export multiple.
  2. Cliquez sur ↓ Exporter la sélection.
  3. Chaque map est générée avec les paramètres actuels et téléchargée automatiquement.

Les fichiers sont nommés automatiquement avec le suffixe correspondant :

Conseils GTA V / FiveM

Nommage des fichiers

GTA V utilise des conventions de nommage strictes :

texture.ytd        → texture principale (diffuse)
texture_n.ytd      → normal map
texture_s.ytd      → specular map

Normal Map — Convention DirectX

✓ Toujours activer
Activez toujours Inverser Y (DirectX) pour les assets GTA V. Le moteur RAGE attend le canal vert inversé par rapport à la convention OpenGL.

Specular Map dans OpenIV

Pour importer dans un .ytd via OpenIV : la specular doit être en format DXT1 (grayscale, sans alpha) ou DXT5 si vous avez besoin d'un canal alpha. L'outil exporte en PNG — convertissez ensuite avec OpenIV ou un outil DDS.
